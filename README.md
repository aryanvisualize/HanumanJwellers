# 💎 Hanuman Jewellers

A modern and elegant **jewellery e-commerce web application** built with a focus on premium UI/UX.
This project showcases a digital storefront for gold, diamond, and gemstone jewellery collections.

---

## ✨ Features

* 🛍️ Browse jewellery by categories:

  * Rings
  * Necklaces
  * Bangles
  * Bracelets
  * Gems & Stones

* 🎯 Clean and minimal UI with luxury design

* 🖼️ High-quality product images

* 🛒 Add to Cart functionality

* 📱 Fully responsive design

* ⚡ Fast performance using modern frontend tools

---

## 🛠️ Tech Stack

* **Frontend:** React.js, Vite
* **Styling:** CSS / Tailwind (if used)
* **State Management:** Context API
* **Version Control:** Git & GitHub

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/aryanvisualize/HanumanJwellers.git
cd HanumanJwellers
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run the project

```bash
npm run dev
```
